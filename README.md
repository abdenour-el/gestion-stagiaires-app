# Systeme de Gestion de Stagiaires (Full Stack)

Bienvenue sur mon projet Full Stack. C'est une application web qui permet de gerer les stagiaires d'une entreprise.

Ce projet connecte une interface React.js (Frontend) avec une API Laravel (Backend).

Technologies Utilisees :

- Frontend : React.js (Vite), Bootstrap 5, Axios, React Router.
- Backend : Laravel 11, MySQL (Base de donnees).

Fonctionnalites Principales (CRUD) :

1. Ajouter un nouveau stagiaire (Nom, Prenom, Email, Filiere).
2. Afficher la liste complete des stagiaires.
3. Modifier les informations d'un stagiaire existant.
4. Supprimer un stagiaire de la base de donnees.
5. Recherche en temps reel par nom ou filiere.

Comment lancer le projet ?

Partie 1 : Backend (Laravel)
- Ouvrez le dossier backend dans le terminal
- Lancez la commande : composer install
- Configurez le fichier .env avec votre base de donnees
- Lancez la commande : php artisan migrate
- Demarrez le serveur : php artisan serve

Partie 2 : Frontend (React)
- Ouvrez le dossier frontend dans le terminal
- Lancez la commande : npm install
- Demarrez le site : npm run dev
